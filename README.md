# **BangumiAPP主要目的是为了圣地巡礼**

📊ANIME的数据来自Bangumi网站，

🗺️圣地巡礼相关的数据来自Anitabi网站。

应用可以查看✅当季新番信息，✅每日新番放送，✅Bangumi番剧排行榜，

新番详情页中可以查看部分番剧现有的圣地巡礼地点，支持我的收藏
